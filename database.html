Notion de Base de Donnée
Définition

Une base de données est une collection de données organisées de façon à être facilement accessibles, administrées et mises à jour.
Analyse et Conception
Analyse

Elle consiste à étudier le problème et à consigner dans un document les besoins, les choix, les contraintes entre autres.

Analyse et Conception
Conception

Langage de programmation SQL
SQL (Structured Query Language)

Langage permettant de communiquer avec une base de données. Ce langage informatique est notamment très utilisé par les développeurs web pour communiquer avec les données d’un site web.

SGBD: définition et exemples
Définition

Un SGBD est un logiciel qui permet de structurer, de stoquer, de mettre à jour et de faire la maintenance d'une base de données.

MYSQL
Définition

MySQL est la base de données open source la plus plébiscitée au monde. De par sa performance, sa fiabilité et sa facilité d’utilisation éprouvées, MySQL s’est imposée comme le choix évident de base de données pour les applications Web, utilisée par les plus grands noms d’Internet, comme Facebook, Twitter, YouTube et les cinq premiers sites Internet*.
